Features:
* Upcoming events with links to Meetup.com
* Social links
* About Us

Built using:
* Xamarin.Forms 3.2
* Prism 7.1
* Autofac
* HttpTracer
* Xamarin.Essentials
* MonkeyCache

Functionality:
* Offline cache for event list
* Icon font used for social icons and tab navigation
* Android bottom tabs
* CSS based styling
* Control Template + Base Page for clean, simple content pages
